## How to Use

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/mandelbrot-exploration.git
   cd mandelbrot-exploration

2. **Install dependencies:**
    ```bash
    pip install -r requirements.txt

3. **Open and run the Jupyter Notebook:**
    ```bash
    jupyter notebook Mandelbrot_Exploration.ipynb


## Important Note

The parameter values provided in the Jupyter Notebook are optimized for GPU programming. If you are running the code on a CPU, be aware that the computation may take a significantly longer time. Consider adjusting the parameters based on your hardware specifications for optimal performance.
